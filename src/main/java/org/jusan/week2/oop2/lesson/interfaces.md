# Интерфейсы в программировании

## Интерфейс
**Интерфейс** (не путать с внешним видом приложения) — это способ описания **абстракции** для объектов.

### Основные особенности интерфейсов:
- **Набор интерфейсов** в проекте представляет собой абстрактный уровень приложения, который всегда необходимо создавать.
- Часто интерфейс называют **"класс без реализации"**.
- Интерфейс:
    - **Не может содержать реализацию**, а только описывает контракт (схему обязанностей), который объекты обязаны выполнить.
    - **Не может быть инстанцирован** — это не класс, а его "фантом".
    - Содержит **абстрактные методы** — методы, которые описаны, но не имеют реализации.

### Интерфейс как "фантом" класса:
Интерфейс можно представить как "фантом" класса, из которого впоследствии появляются реальные объекты.

---

## Реализация интерфейса: `implements`
Ключевое слово **`implements`** используется в классе, который хочет реализовать интерфейс.

### Обязательства класса:
- Класс **обязан реализовать все методы интерфейса**, иначе компилятор выдаст ошибку.
- Про класс говорят: **"реализует интерфейс"** или **"имплементирует интерфейс"**.

### Рекомендации по именованию:
- В начале имени интерфейса часто добавляют заглавную букву **`I`** (например, `IReadable`), чтобы сразу было понятно, что это интерфейс.
- Иногда используют нотацию в виде английских прилагательных (например, `Movable`), но добавление "I" считается более наглядным.

### Наследование:
- Нельзя унаследоваться от интерфейса как от обычного класса с помощью `extends`. Интерфейс можно только **реализовать** с помощью `implements`.

---

## Аннотация `@Override`
При реализации методов интерфейса в конкретном классе используется аннотация **`@Override`**.

### Зачем это нужно:
- Аннотация помогает избежать ошибок на этапе компиляции.
- Указывает, что метод **переопределён** или **реализован** из интерфейса.

---

## Особенности интерфейсов:
1. **Методы:**
    - Автоматически имеют модификатор доступа **`public`**.
    - Не могут иметь тела (без фигурных скобок).

2. **Переменные:**
    - В интерфейсе нельзя объявлять переменные (свойства) — только **константы**.
    - Все константы автоматически **`public static final`**.
